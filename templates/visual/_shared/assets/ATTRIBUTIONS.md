# Visual Asset Attributions

The shared visual asset set is fully local and offline.

- EFP generic visual symbols: original local vector shapes for this repository.
- AWS service semantic placeholders: local generic cloud-service glyphs for `aws.*` icon IDs. These are not official AWS Architecture Icons and do not use AWS logo artwork.
- Jenkins CI semantic placeholder: local generic CI glyph for the `jenkins` icon ID. This is not Jenkins project artwork and does not use the Jenkins logo.

No remote assets are loaded at render time.
